# fixtures/skills

Hand-crafted **golden** `SKILL.md` + `skill.meta.json` pairs for each of the
five reference workflows listed in CLAUDE.md:

- `gmail_reply`
- `calendar_block`
- `finder_organize`
- `slack_status`
- `notes_daily`

These are the ground-truth outputs the synthesizer's snapshot similarity
test (S-014) and real-mode smoke test (S-017) compare against.

## Do NOT regenerate these automatically

Ralph iterations and other automated agents **must not** overwrite these
files from a synthesized draft. The whole point of a snapshot test is that
the comparison baseline is independent of the system under test — if the
golden is generated by the same pipeline being evaluated, the test becomes
tautological and stops catching regressions.

When something legitimately changes (the SKILL.md format evolves, a new
reference workflow is added, a parameter name convention shifts), update
these files **by hand**, commit them alongside the change that required
them, and re-run `scripts/check_fixtures.sh`.

## Invariants every fixture satisfies

Enforced by `tests/synthesizer/test_golden_fixtures.py` and
`scripts/check_fixtures.sh`:

- `skill.meta.json` validates against `contracts/skill-meta.schema.json`.
- `SKILL.md` round-trips through the S-003 parser:
  `parse_skill_md(render_skill_md(parse_skill_md(f))) == parse_skill_md(f)`.
- `validate_meta_against_markdown` agrees on step count, destructive flags,
  and parameter references.
- At least four steps; at least one parameter.
- At least one destructive step for `gmail_reply`, `finder_organize`, and
  `slack_status` (their workflows are irreversible once committed).
  `calendar_block` and `notes_daily` may declare zero destructive steps.
- `meta.trajectory_id` matches the `id` field in the corresponding
  `fixtures/trajectories/<dir>/metadata.json`.
- Total size of `fixtures/skills/` stays under 1 MB — screenshots belong in
  `fixtures/trajectories/`, not here.

## Validating the fixtures

```shell
./scripts/check_fixtures.sh            # < 5 s; exits non-zero on any failure
uv run pytest tests/synthesizer/test_golden_fixtures.py
```
