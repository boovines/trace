# Calendar Block

Open Google Calendar and create a focus block tomorrow at a given time for a
given duration.

## Parameters

- `start_time` (string) — Start time, e.g. `2:00 PM`.
- `duration_minutes` (number) — Length of the block in minutes.

## Steps

1. Open Google Chrome and navigate to https://calendar.google.com.
2. Click the Create button in the top-left corner.
3. Choose Event from the dropdown.
4. Set the title to "Focus block".
5. Set the start time to {start_time} tomorrow.
6. Set the duration to {duration_minutes} minutes.
7. ⚠️ Click Save to create the event.
