# Notes Daily

Open Apple Notes and create a new note titled with today's date, pasting a
fixed template body.

## Parameters

- `template_body` (string) — Note body text.

## Steps

1. Open the Apple Notes app.
2. Click the "New Note" button in the toolbar.
3. Type today's date as the note title and press Return.
4. Paste {template_body} into the note body.
