# Slack Status

Open Slack and set the status to a given emoji + message with a timed expiry.

## Parameters

- `emoji` (string) — The status emoji short code, e.g. `:dart:`.
- `message` (string) — Status message text.
- `expiry` (string) — Expiry label, e.g. `2 hours`.

## Steps

1. Open the Slack app.
2. Click the profile avatar in the top-right corner.
3. Click "Update your status" in the menu.
4. Type {emoji} into the emoji picker and select the match.
5. Type {message} into the status text field.
6. Set the expiry dropdown to {expiry}.
7. ⚠️ Click Save to apply the status.
